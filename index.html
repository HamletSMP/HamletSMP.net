Important Notice

Hamlet SMP is currently in PRE-ALPHA (0.1) development. This means the server is in early testing phase and not yet ready for public gameplay. We are actively recruiting developers, plugin creators, and dedicated staff to help build this revolutionary SMP experience from the ground up.

The server will experience frequent restarts, downtime, and bugs as we develop our unique vision for Minecraft multiplayer. We're building something truly special here, and we need talented individuals to help bring this ambitious project to life.

Development Vision

Revolutionary Concept

We're developing a completely new kind of SMP experience with mechanics never before seen in Minecraft multiplayer. More details will be revealed to our development team.

Recruiting Staff

We're looking for dedicated developers, plugin creators, and moderators to join our team. If you have experience with Java, Spigot/Paper, or server management, we need you!

Unique Gameplay

Our server will feature proprietary gameplay systems that fundamentally change how players interact with the world and each other. This isn't another basic SMP clone.

In Development

Core systems are currently being designed and implemented. We're building the foundation for what will become a groundbreaking Minecraft server experience.

Development Timeline

January 2026: Hamlet SMP development officially began. Core concept finalized, server domain (hamletsmp.net) acquired, and initial infrastructure planning completed. Plugin testing phase initiated.
